Positive mammogram
  1. ASSUME breast-cancer (bernoulli 0.01)
  2. ASSUME positive-mammogram (lambda () (if breast-cancer (bernoulli 0.8) (bernoulli 0.096)))
  3. OBSERVE (positive-mammogram) true
  4. INFER 1000
  5. PREDICT (positive-mammogram)

Fair/tricky coin
  1. ASSUME isTricky (bernoulli 0.1)
  2. ASSUME myCoinWeight (if isTricky (beta 1.0 1.0) 0.5)
  3. ASSUME myCoin (lambda () (bernoulli myCoinWeight))
  4. OBSERVE (myCoin) true (repeat 9 times)
  5. INFER 1000
  6. PREDICT isTricky
  7. INFER 1000
  8. PREDICT isTricky

Sprinkler Bayes network
  1. ASSUME cloudy (bernoulli 0.5)
  2. ASSUME sprinkler (bernoulli (if cloudy 0.1 0.5))
  3. ASSUME rain (bernoulli (if cloudy 0.8 0.2))
  4. ASSUME wet-grass (bernoulli (if sprinkler (if rain 0.99 0.9) (if rain 0.9 0.0)))
  5. OBSERVE sprinkler true
  6. PREDICT rain

Ising
  1. ASSUME x1-1 (bernoulli)
  2. ASSUME x1-2 (bernoulli)
  3. ASSUME x2-1 (bernoulli)
  4. ASSUME x2-2 (bernoulli)
  5. OBSERVE (bernoulli (power r[2.71828] (if (= x1-2 x1-1) r[0] r[-1]))) true
  6. OBSERVE (bernoulli (power r[2.71828] (if (= x2-1 x1-1) r[0] r[-1]))) true
  7. OBSERVE (bernoulli (power r[2.71828] (if (= x2-2 x1-2) r[0] r[-1]))) true
  8. OBSERVE (bernoulli (power r[2.71828] (if (= x2-2 x2-1) r[0] r[-1]))) true

Simple PCFG
  1. ASSUME out (lambda () (normal (if (bernoulli 0.7) 1 (+ (out) (out))) 2.0))
  2. PREDICT (out)
